# branches

`git checkout -b <new_branch> <start_branch>`  
e.g. `git checkout -b develop master`  

`git branch -d <branch_to_delete>`  
`git branch origin :<branch_to_delete>` to delete remote branch  

`git push --all`  

`git log --graph --decorate --oneline`
